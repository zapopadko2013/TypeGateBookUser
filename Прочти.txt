1. Чтобы запускать программы не в докере нужно для каждой программы 
установить необходимые библиотеки войдя в папку программы и выполнив
команду:

  npm install
2. После это нужно запустить каждую программу войдя в папку программы и выполнив
команду:

 npm run start
3. После этого можно подавать необходимые запросы, мутации. Для этого в браузере
нужно войти по адресу и перейти на сайт выполнения запросов:

http://localhost:4000/graphql

4. Для запуска программ в докере нужно выполнить команду из основной папки:

docker-compose up
